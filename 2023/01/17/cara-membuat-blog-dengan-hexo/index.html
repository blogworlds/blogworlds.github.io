cara membuat blog dengan hexo

Hexo adalah platform pembuatan blog yang populer dan mudah digunakan. Ini adalah pilihan yang baik bagi Anda yang ingin membuat blog pribadi atau profesional tanpa harus mengeluarkan banyak waktu dan usaha. Dalam panduan ini, kami akan menunjukkan kepada Anda cara membuat blog dengan Hexo dari awal.

Langkah 1: Instal Node.js dan Git

Sebelum memulai, Anda harus memiliki Node.js dan Git terinstal di komputer Anda. Node.js adalah platform yang digunakan oleh Hexo untuk berjalan, sementara Git digunakan untuk mengelola versi dari blog Anda. Anda dapat mengunduh Node.js dan Git dari situs web resmi masing-masing.

untuk termux dapat menggunakan perintah:
nodejs
pkg install nodejs
git
pkg install git
