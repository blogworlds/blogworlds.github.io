belajar html part2

Pada bagian kedua ini, kita akan belajar lebih dalam tentang elemen-elemen HTML yang digunakan dalam pembuatan halaman web.

  1. Headings
    Elemen headings digunakan untuk memberikan judul dan subjudul pada halaman web. Ada enam level heading yang tersedia dalam HTML, yaitu <h1> sampai <h6>. <h1> adalah heading level tertinggi yang digunakan untuk judul utama halaman, sementara <h2> sampai <h6> digunakan untuk subjudul.
    contoh:
  2. Paragraf
    Elemen <p> digunakan untuk membuat paragraf pada halaman web. Elemen ini akan menambahkan jarak spasi di atas dan di bawah paragraf yang dibuat.
    contoh:
  3. Tautan
    Elemen <a> digunakan untuk membuat tautan pada halaman web. Atribut “href” digunakan untuk menentukan alamat URL yang dituju oleh tautan.
    Contoh:
  4. Gambar
    Elemen <img> digunakan untuk menambahkan gambar pada halaman web. Atribut “src” digunakan untuk menentukan lokasi gambar yang akan ditampilkan. Anda juga dapat menambahkan atribut “alt” untuk memberikan deskripsi gambar yang akan ditampilkan saat gambar tidak dapat ditampilkan.
    contoh:
  5. Daftar
    Elemen <ul> digunakan untuk membuat daftar bullet, sedangkan elemen <ol> digunakan untuk membuat daftar numerik. Elemen <li> digunakan untuk menambahkan item dalam daftar.
    contoh:
  6. Div dan Span
    Elemen <div> digunakan sebagai container untuk elemen HTML lainnya. Elemen ini sering digunakan untuk membuat bagian-bagian halaman yang digunakan sebagai container untuk elemen HTML lainnya. Sementara elemen <span> digunakan untuk membuat bagian-bagian teks yang digunakan sebagai container untuk elemen HTML lainnya.
    contoh:

Itulah beberapa elemen HTML yang sering digunakan dalam pembuatan halaman web. Selain elemen-elemen tersebut, masih banyalah elemen-elemen lain yang bisa anda pelajari untuk membuat halaman web yang lebih kompleks dan interaktif. Namun dengan menguasai elemen-elemen dasar tersebut anda sudah dapat membuat halaman web yang fungsional dan informatif. Selamat belajar dan semoga sukses dalam pembuatan halaman web Anda!
